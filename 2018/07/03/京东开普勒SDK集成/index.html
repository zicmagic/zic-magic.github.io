细节补充

1、京东有些页面会出现点击下载的空间，拦截也需要注入js，处理方式和前面一片描述的相同

2、京东会出现以下两种掉单情况：（1）后端接口能查询到购买商品，京东后台无法查到购物商品（2）后台订单中有订单，但是找不到对应的购买人ID。为了解决此问题也需要像淘宝的一样，后台拉取用户订单列表。
